# AMAZON
![](/demo.gif)

Fullstack online store with the ability to pay for goods through the "Yookassa" service
## Getting start
### Start back-end:
``` 
cd server-side  
```
``` 
npm i
```
``` 
npm start dev
```

### Start front-end:
``` 
cd client-side 
```
``` 
npm i
```
``` 
npm run dev
```


## Stack
### Front-end:
- React
- Next JS
- TypeScript
- Redux TLK
- React Query

### Back-end:
- Node.js
- NestJS
- PostgreSQL
